Numerical Integration Methods

Using Numerical Integration Methods to Calculate Constants and Uncertainty Values in Quantum Mechanics on Jupyter Notebook

This was the project I carried out as a part of my university coursework. It involves me using three numerical integration methods (Trapezium rule, Simpson's Rule and Monte-Carlo integration) to calculate the expectation values of position and their uncertainties. Then finding the unknown constants in the wavefunctions. The aim was to find the best integration method for the all 3 parts of the wave function. Project carried out using Python on Jupyter Notebook.
